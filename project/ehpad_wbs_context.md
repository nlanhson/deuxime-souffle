# EHPAD Manager (P02) — WBS Scope Extract (authoritative)

> Persona P02 = Nursing Home Manager. Non-technical (director/activities manager / animatrice).
> Accesses via responsive web (tablet-first; also desktop + mobile browser). V1 = WEB ONLY (EHPAD mobile app is V2).
> Several people may share one EHPAD account (high staff turnover). Two roles: Admin (Main Contact) / User (Other Contact).
> Source: WBS '2_Stories' sheet, rows where Role includes Nursing Home Manager / EHPAD.

## EHPAD sidebar navigation (from E05 / EHPAD video):
Home (Dashboard) · Sessions · Rate sessions · Contracts · Contacts · Invoices · Your information (facility profile) · Contact us · Notifications.

### AUTH-04 — EHPAD Account Activation via Invite Link  _(epic: Authentication & Account Management; role: Nursing Home Manager)_

User can enter password and confirm password. ● Password and confirm password must match. ● Password must meet security rules. ● Expired or invalid link shows error message. ●. Successful submission activates account
**Acceptance criteria:** As an EHPAD Manager, I want to create and activate my account from an invitation link so that I can access and manage my establishment. AC1. User can access account setup through a valid invitation link. AC3. Email field shall be prefilled from the invitation and cannot be edited. AC4. User shall provide: first name last name number phone roles password confirm password AC5. Password and confirm password must match and follow system password rules. AC6. Invitation link can only be used once. AC7. If invitation link is expired, invalidated, or already used, system shall prevent access and display an appropriate message. AC8. Successful submission activates the account and links the user to the corresponding establishment. AC9. After successful activation, user can log in and access the platform according to assigned permissions.

### AUTH-07 — Password Reset  _(epic: Authentication & Account Management; role: APA Coach, Nursing Home Manager, DS Admin)_

**Acceptance criteria:** As a user (APA Coach, EHPAD Manager, DS Admin), I want to reset my password if forgotten so that I can regain access to my account. AC1. User shall be able to request a password reset by entering their email address. - If email field is empty, system shall display error: "This field is required" and prevent submission. AC2. Email must be in valid format. - If email format is invalid, system shall display error: "Invalid email format" and prevent submission. AC3. When user submits a valid email: - System shall send a password reset link to the provided email address. AC4. Password reset link shall be time-limited. - If the link is expired (after 1 hour), system shall display error: "Link expired" and require user to request a new link. AC5. When user clicks a valid reset link: - System shall open the Reset Password screen. AC6. Reset Password screen shall allow user to enter: - New password - Confirm password AC7. New password must meet security requirements: - Minimum 8 characters - At least 1 uppercase letter - At least 1 number - If not met, system shall display error: "The password does not meet the security rules" and prevent submission. AC8. Confirm password must match new password. - If not matched, system shall display error: "Passwords do not match" and prevent submission. AC9. When user submits a valid new password: - System will update the password successfully. AC10. After successful password reset: - System shall display confirmation message: "Password successfully reset" - User shall be redirected to login screen. AC11. If reset link is invalid or already used: - System shall display error: "Link invalid or already used".
