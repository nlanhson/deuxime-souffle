# Feedback drop zone

Drop coach-app (or EHPAD) feedback here and tell Claude "implement the feedback in project/feedback/".

Any format works:

- **Markdown / text** — a simple numbered list is ideal (one item per change)
- **Screenshots** — annotated PNGs/JPGs are fine; name them after the screen (e.g. `dashboard-spacing.png`)
- **PDF / docx** — exported notes from the client are fine too

Tips for fast, accurate implementation:

- Say **which screen** each point applies to (e.g. "Sessions list", "Athlete profile")
- Mark priority if it matters: `[must]` / `[nice]`
- If something should *not* change, say so explicitly
