Logo — the real Deuxième Souffle brand mark (the "2ᵉᵐᵉ" picto tile, customer-supplied SVG).

The artwork is a solid tile with the "2EME" mark knocked out, so the mark shows the background
through it. The tile fill is the single controllable colour: dark ink (BLACK RING #181715) on the
light cream canvas — the default now that coach is light — and white when it sits over a photo or
dark layer (the Welcome hero passes color="#FFFFFF"). Single source for the
app logo so the splash, the welcome lockup, and the login header all render the same asset at
different sizes. Clipped to the design-system radius so it reads as a rounded brand badge.
Decorative by default — it sits next to the wordmark/title that already carry the brand name.
